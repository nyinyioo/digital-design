## Overview

This repository contains my digital design work in SystemVerilog, 
including RTL practice (HDLBits), a VGA display controller, 
an SPI implementation of the VGA modules, and a FIFO buffer. 
These projects focus on FSMs, timing, memory, and writing testbenches.

## 

| Project | Description | Status |
|---|---|---|
| HDLBits | FSM, sequential, combinational problems | ✅ Completed |

| SPI | VGA modules over SPI protocol | ⚙️ In Progress |
| FIFO | Synchronous FIFO buffer | ⚙️ In Progress |
